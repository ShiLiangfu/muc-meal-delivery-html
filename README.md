# 校园外卖平台 - 纯前端版本

这是校园外卖平台的纯前端版本，可以直接通过浏览器打开HTML文件运行，无需安装Node.js或运行npm命令。

## 功能特性

- ✅ 首页展示和导航
- ✅ 商家列表和搜索
- ✅ 商家详情和菜单展示
- ✅ 跑腿服务发布和查看
- ✅ 用户登录注册（本地存储）
- ✅ 个人中心和订单管理
- ✅ 响应式设计，支持移动端
- ✅ 完整的UI交互和动画效果

## 使用方法

### 方法一：直接打开
1. 双击 `index.html` 文件
2. 浏览器会自动打开应用

### 方法二：本地服务器（推荐）
如果遇到跨域问题，建议使用本地服务器：

```bash
# 使用Python（如果已安装）
python -m http.server 8000

# 使用Node.js（如果已安装）
npx serve .

# 使用PHP（如果已安装）
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 技术栈

- **Vue 3** - 渐进式JavaScript框架
- **Element Plus** - Vue 3组件库
- **原生CSS** - 自定义样式和响应式设计
- **本地存储** - 用户状态持久化

## 数据说明

由于这是纯前端版本，所有数据都是模拟数据：

- 商家信息：预设了4个示例商家
- 菜单数据：每个商家都有示例菜品
- 用户数据：存储在浏览器本地存储中
- 订单数据：模拟的订单记录

## 注意事项

1. **网络连接**：需要网络连接来加载CDN资源（Vue、Element Plus等）
2. **本地存储**：用户登录状态保存在浏览器本地存储中
3. **图片资源**：确保images文件夹完整，否则图片可能无法显示
4. **跨域问题**：如果直接打开HTML文件遇到问题，请使用本地服务器

## 与原项目的区别

| 特性 | 原项目 | 纯前端版本 |
|------|--------|------------|
| 运行方式 | npm run dev | 直接打开HTML |
| 依赖管理 | npm/yarn | CDN引入 |
| 构建工具 | Vite | 无需构建 |
| 路由 | Vue Router | 单页面切换 |
| 状态管理 | Pinia | 本地状态 |
| 数据存储 | API调用 | 本地存储 |
| 部署方式 | 需要构建 | 直接复制文件 |
