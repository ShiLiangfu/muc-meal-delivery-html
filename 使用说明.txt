校园外卖平台 - 纯前端版本使用说明
=====================================

🎉 欢迎使用校园外卖平台纯前端版本！

📋 快速开始
----------
1. 双击 "start-server.bat" 文件，选择启动方式
2. 或者直接双击 "index.html" 文件
3. 在浏览器中体验完整功能

✨ 主要功能
----------
• 首页：平台介绍和功能导航
• 商家列表：浏览和搜索商家
• 商家详情：查看菜单和商品
• 购物车系统：添加商品、计算金额、管理数量
• 订单确认：完整的下单流程和订单管理
• 跑腿服务：发布任务、接单功能
• 用户系统：注册、登录、个人中心
• 订单管理：查看订单历史和状态

🔧 技术特点
----------
• 基于Vue 3 + Element Plus
• 无需安装Node.js或npm
• 所有依赖通过CDN加载
• 数据存储在浏览器本地
• 完全响应式设计

📱 测试账号
----------
您可以注册新账号，或使用以下测试数据：
• 用户名：随意输入
• 密码：随意输入
• 所有数据都是模拟数据，仅用于演示

🌐 浏览器要求
------------
• Chrome 88+ (推荐)
• Firefox 85+
• Safari 14+
• Edge 88+
• 需要网络连接加载CDN资源

❗ 注意事项
----------
1. 首次打开可能需要几秒钟加载CDN资源
2. 用户数据保存在浏览器本地存储中
3. 清除浏览器数据会丢失登录状态
4. 如遇到问题，请使用本地服务器方式启动

📞 问题反馈
----------
如果遇到任何问题，请检查：
1. 网络连接是否正常
2. 浏览器是否支持
3. 是否有安全软件阻止
4. 尝试使用不同的启动方式

🎯 与原项目对比
--------------
原项目需要：npm install → npm run dev
纯前端版本：直接打开 → 立即使用
