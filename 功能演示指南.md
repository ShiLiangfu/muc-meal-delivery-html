# 校园外卖平台 - 功能演示指南

## 🎯 完整功能演示流程

### 1. 用户注册和登录
1. 打开应用，点击右上角"注册"
2. 填写用户名、昵称、密码
3. 注册成功后自动登录
4. 或使用"登录"功能登录已有账号

### 2. 浏览商家和菜单
1. 点击"商家列表"查看所有商家
2. 使用搜索框搜索特定商家或菜品
3. 点击商家卡片进入商家详情页
4. 浏览菜单，查看商品详情和价格

### 3. 购物车功能演示 ⭐
1. 在商家详情页面，点击商品的"加入购物车"按钮
2. 右上角会显示购物车图标和商品数量
3. 点击"购物车"按钮打开购物车
4. 在购物车中可以：
   - 调整商品数量
   - 删除不需要的商品
   - 查看商品小计和配送费
   - 检查是否达到起送金额

### 4. 下单流程演示 ⭐
1. 在购物车中点击"去结算"
2. 进入订单确认页面，显示：
   - 商家信息和地址
   - 订单详情（商品、数量、价格）
   - 费用明细（商品小计 + 配送费）
   - 配送信息
3. 点击"确认下单"完成订单
4. 订单自动添加到"我的订单"中

### 5. 跑腿服务演示 ⭐
1. 点击"跑腿服务"进入跑腿页面
2. 查看当前可接的跑腿任务
3. 点击"发布跑腿任务"发布新任务：
   - 填写任务标题和描述
   - 设置报酬金额
   - 选择截止时间
4. 对于待接单的任务，点击"接单"按钮
5. 接单后任务状态变为"已接单"，显示接单者信息

### 6. 个人中心和订单管理
1. 点击用户头像下拉菜单
2. 选择"个人中心"查看个人信息
3. 选择"我的订单"查看所有订单
4. 订单显示详细信息和状态

## 💡 使用技巧

1. **数据持久化**：用户登录状态会保存在浏览器中
2. **购物车状态**：购物车内容在页面刷新后仍然保留
3. **多商家购物**：每次只能从一个商家购买，切换商家会清空购物车
4. **任务接单**：需要登录后才能接单
5. **订单查看**：所有订单都会保存在本地，可随时查看
