# 校园外卖平台 - 纯前端版本

## 🎯 项目概述

这是基于原Vue.js校园外卖项目创建的纯前端版本，无需安装Node.js或运行npm命令，可以直接通过浏览器打开使用。

## 🚀 快速体验

### 方法一：一键启动（推荐）
```
双击 "start-server.bat" → 选择启动方式 → 浏览器自动打开
```

### 方法二：直接打开
```
双击 "index.html" → 浏览器打开应用
```

## 🛠 技术架构

```
前端框架：Vue 3 (Composition API)
UI组件：Element Plus 2.3.8
样式方案：原生CSS + CSS变量
图标库：Element Plus Icons
数据存储：LocalStorage
资源加载：CDN (无需本地安装)
```

## 🚀 部署方式

### 本地部署
1. 解压文件到任意目录
2. 双击启动脚本或HTML文件
3. 浏览器自动打开应用

### 服务器部署
1. 上传所有文件到Web服务器
2. 配置静态文件服务
3. 通过域名访问应用

### CDN部署
1. 上传到CDN服务商
2. 配置缓存策略
3. 全球加速访问
